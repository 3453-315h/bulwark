Bulwark is an asset and vulnerability management tool.

## Installation

Use the package manager [npm](https://www.npmjs.com/get-npm) to install Bulwark.

```bash
npm install
```

## Usage

### Production
```bash
npm start
```
### Development
```bash
npm run start:dev
```
